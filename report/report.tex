\begin{abstract}
CIFAR-100 from scratch (compact SE-ResNet): Stage~A $\rightarrow$ Stage~B (\textbf{65.17\%} test) $\rightarrow$ Stage~C with Mixup/CutMix/EMA (\textbf{68.14\%} test). Evaluation uses \texttt{results.py}.
\end{abstract}

\section{Introduction}
We train on CIFAR-100 without pretrained weights (Colab; checkpoints on Drive) and report test accuracy via the course \texttt{results.py} script.

\section{Dataset}
CIFAR-100: $50{,}000$ train / $10{,}000$ test. Per-channel normalization uses $\mu=(0.5071,0.4867,0.4408)$ and $\sigma=(0.2675,0.2565,0.2761)$. Stages~B/C hold out $10\%$ stratified validation (seed $42$); Stage~A uses all training images and evaluates on the test set each epoch.

\section{Methodology}
\subsection{Model Architecture}
\textbf{SE-ResNet} (\texttt{se\_resnet}): SE attention, \texttt{base\_width}${}=32$, two blocks per stage (\texttt{2$\times$2$\times$2$\times$2}), GAP, linear to $100$ classes.

\subsection{Training Strategy}
\begin{itemize}[leftmargin=1.2em, itemsep=0.15em, topsep=0.25em]
    \item \textbf{Stage A:} $4$ epochs, basic aug, SGD lr $0.1$, warmup $1$, cosine; test each epoch; $\sim$33\% checkpoint for init.
    \item \textbf{Stage B:} \texttt{--init-from}; RandAugment ($N{=}2$, $M{=}9$), erasing $0.05$, label smooth $0.05$, SGD lr $0.05$, warmup $5$, $30$ epochs; val for selection; test via \texttt{results.py}. No EMA.
    \item \textbf{Stage C:} From Stage~B \texttt{best.pt}, $30$ epochs (\path{scripts/iterate_from_stageb_stage_c_mixema.sh}), RandAugment, Mixup/CutMix, lr $0.015$, warmup $3$, batch $128$, clip $1$, EMA $0.999$. Select by val accuracy.
\end{itemize}
Code: \texttt{train.py}, \texttt{model.py}, \texttt{utils.py}; Stages~A--B via \path{scripts/quick_se_resnet_results.sh}; Stage~C via Colab/\path{scripts/iterate_from_stageb_stage_c_mixema.sh}. AMP on CUDA. Best test from \texttt{results.py} on Stage~C \texttt{best.pt}.

\section{Discussion}
\begin{itemize}[leftmargin=1.2em]
    \item \textbf{Initialization matters:} Starting from the Stage~A weights gave a large jump over training the finetune recipe from scratch in the same wall-clock budget.
    \item \textbf{RandAugment + longer schedule:} Stronger augmentation and more epochs on top of the Stage~A baseline outperformed a larger Wide-ResNet trained from scratch in the same time budget.
    \item \textbf{Stage~C on top of Stage~B:} Treating the $\sim$65\% model as a \emph{base} checkpoint and finetuning with Mixup, CutMix, and EMA at a lower learning rate yielded $\sim$3 percentage points on official test ($65.17\% \rightarrow 68.14\%$), with best validation $68.48\%$.
    \item \textbf{Trade-offs:} The SE-ResNet path is faster to iterate than a full Wide-ResNet Hydra ladder; the cost is a lower ceiling than very large models trained for hundreds of epochs on CIFAR-100.
\end{itemize}

\section{Conclusion}
We trained a compact SE-ResNet on CIFAR-100 (Stages~A--B), then improved it with Stage~C (Mixup, CutMix, EMA) from the Stage~B checkpoint. The best official test accuracy is \textbf{68.14\%} (\texttt{results.py}), with Stage~B alone at \textbf{65.17\%}. Future work could use longer schedules or wider models if compute permits.
